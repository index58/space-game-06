# Описание игрового мира.

## Общее
Компьютерная игра.
Браузерная.
Многопользовательская онлайн ПВЕ/ПВП.
2D. Космос. Вид сверху.
Пространство космоса бесконечно во все стороны.
Основную камеру можно приближать (zoom in) и удалять (zoom out).
Если игрок заходит в игру первый раз, то он создаёт себе персонажа (указывает никнейм).
В каждый момент времени игрок может играть от лица только одного из своих персонажей.
Для своего нового персонажа игрок выбирает стартовый космический корабль (один из самых маленьких).
Выбранный стартовый космический корабль достаётся новому персонажу даром (становится его собственностью).

## Нахождение персонажа внутри космического объекта
Персонаж в игре всегда находится в каком-то одном космическом объекте.
Самого персонажа не видно и не слышно в игре нигде и никогда, он проявляется только своим ником и сообщениями в чате.

Существует 2 вида космических объектов, внутри которых может находиться персонаж:
- Космический корабль
- Космическая станция

Внутри собственного космического объекта персонаж может находиться:
- В кабине пилота
- В хозяйственном отсеке
- В пассажирском кресле

Внутри чужого космического объекта персонаж может находиться:
- В пассажирском кресле

### Кабина пилота
Кабина пилота есть только в космическом корабле.
В кабине пилота можно пилотировать космический корабль (управлять его движением).
Основная игровая камера следует за пилотируемым кораблём так,
что корабль всегда находится в центре нижней половины экрана носом строго вверх экрана.

В кабине пилота есть панель пилота, в которой доступно:
- Стрельба из оружия космического корабля
- Подбор лута с помощью специального оборудования космического корабля
- Добыча ресурсов с помощью специального оборудования космического корабля
- Сканирование пространства радаром космического корабля

### Хозяйственный отсек
Хозяйственный отсек есть и в космическом корабле и в космической станции.
Основная игровая камера расположена так, что космический объект всегда находится в центре экрана.

В хозяйственном отсеке доступно:
- Управление оборудованием через панель управления оборудованием
- Строительство объектов с помощью роботов
- Производство предметов на специальном оборудовании с помощью роботов
- Ремонт объектов с помощью роботов
- Демонтаж объектов с помощью роботов
- Установка оборудования на объект
- Изучение технологий производства и технологий строительства (нужна лаборатория)

### Пассажирское кресло
Пассажирское кресло может быть только в космическом корабле.
Основная игровая камера расположена так, что космический объект всегда находится в центре экрана.

## Пилотирование космического корабля
Космический корабль приводится в движение силами тяги двигателей этого корабля.

У каждого двигателя есть 6 сил тяги:
- Линейная сила тяги влево
- Линейная сила тяги вправо
- Линейная сила тяги вниз
- Линейная сила тяги вверх
- Крутящий момент против часовой стрелки
- Крутящий момент по часовой стрелке

Все линейные силы тяги одного двигателя имеют одинаковые значения по модулю, все его крутящие моменты тоже.
Включенная сила тяги работает на 100%, выключенная — на 0%.

Управление движением корабля происходит с помощью команд, включающих и выключающих силы тяги двигателей этого корабля.
* Смещение:
	- Начало смещения влево (включает линейную силу тяги влево у всех двигателей)
	- Конец смещения влево (выключает линейную силу тяги влево у всех двигателей)
	- Начало смещения вправо (включает линейную силу тяги вправо у всех двигателей)
	- Конец смещения вправо (выключает линейную силу тяги вправо у всех двигателей)
	- Начало смещения вниз (включает линейную силу тяги вниз у всех двигателей)
	- Конец смещения вниз (выключает линейную силу тяги вниз у всех двигателей)
	- Начало смещения вверх (включает линейную силу тяги вверх у всех двигателей)
	- Конец смещения вверх (выключает линейную силу тяги вверх у всех двигателей)
* Поворот:
	- Начало поворота влево (включает крутящий момент против часовой стрелки у всех двигателей)
	- Конец поворота влево (выключает крутящий момент против часовой стрелки у всех двигателей)
	- Начало поворота вправо (включает крутящий момент по часовой стрелке у всех двигателей)
	- Конец поворота вправо (выключает крутящий момент по часовой стрелке у всех двигателей)

Шести указанным выше силам соответствуют шесть пользовательских настроек:
- Смещение влево
- Смещение вправо
- Смещение вниз
- Смещение вверх
- Поворот влево
- Поворот вправо

Каждому из этих действий может быть назначено (либо):
- Клавиша, при нажатии на которую происходит начало этого действия, а при отжатии — конец этого действия
- Движение мыши, при котором происходит начало этого действия, а при истечении 0.1 секунды с момента последнего такого движения мыши — конец этого действия

## Единицы измерения
- Расстояние, длина: `Метры` (километры)
- Время: `Секунды` (миллисекунды, минуты, часы, дни)
- Сила: `Ньютоны`
- Скорость: `Метры в секунду`
- Ускорение: `Метры в секунду^2`
